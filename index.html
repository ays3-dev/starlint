What it checks

Spacing Snapper

Flags every element where margin or padding isn't a multiple of 8 px — the industry‑standard grid. Highlights them in red so you can't miss them.

Contrast Guardian

Uses the WCAG luminance formula L = 0.2126R + 0.7152G + 0.0722B to check if text is readable. Flags anything below a 4.5 : 1 ratio.

Ghost Layout Mode

Toggles a pink 1 px outline on every element so you can spot nesting problems, overlapping boxes, and sneaky layout bugs instantly.

Accessibility Check

Finds images missing alt text and form inputs without labels — the most common accessibility mistakes on the web.

Live Report Panel

A real‑time summary panel pops up directly on the page: spacing issues, contrast fails, and accessibility gaps — all counted and listed.

Export Report

Exports the full audit as a JSON file and opens a notepad view — so everyone can read the errors, not just devs.
